Asteroids.jar

(requires class file 59.0 or later/Java version 15 or later)

Polymorphism: This program was designed with Object Oriented Design Principles in mind. for example, within the AsteroidsGame class, an update method essentially calls the update methods of all the AnimatedElement objects. This allows for a polymorphic response where each game element can update itself in its own specialized way. further behavior information of this method can be found in the update methods for each AnimatedElement.

  1.Std Draw: This program utilizes StdDraw, a class belonging to a publically provided Java library called StdLib.

    It provides methods for creating drawings which are created rapidly to create the game animations. Each time the 'game' is updated, the coordinates and other attributes of each GameElement object are updated on the screen.

    By using this publicly availible library, I was able to place more focus on the game behavior.
