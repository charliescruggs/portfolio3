Asteroids

The majority of this code was written by me, with a few exceptions as follows:

  1. GameDriver (main class) was written by my professor.

    • Its purpose was to specify requiremnts to be satisfied through other methods and classes

  2. StdDraw is a program I used which is publicly provided by Princeton University.

    • The StdDraw class provides a basic capability for creating drawings with your programs

This program utilizes Poymorphism by updating each game element in its own specialized way. Within the AsteroidsGame class, an update method essentially calls the update methods of all the updateElements objects. This allows for a polymorphic response where each game element can update itself in its own specialized way. further behavior information of this method can be found in the update methods for each game element.
